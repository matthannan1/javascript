Recursion

Recursive function: calls itself from within itself.
Takes int n
If n = 0, return true (checking for evenness)
If n = 1, return false.
If n < 0, return -n. That is, if -1, return 1, so false.
Final else does the heavy lifting.
Subtracts 2 from n and returns the result, over and over again,
until n is either 0 or 1, and returns true or false.
